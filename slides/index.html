La boucle DO-WHILE

La boucle do-while est une boucle while où les instructions du corps sont exécutées avant de tester la condition de la boucle. 

do
{
    suiteInstructions
}
while (condition);

où condition est une expression booléenne.

  1. suiteInstructions est exécuté.
  2. condition est évaluée à la fin de chaque itération : si elle est vraie, le contrôle revient à suiteInstructions (point 1)
  3. si elle est fausse, le contrôle du programme passe à l’instruction immédiatement après la boucle
L’intérêt d’une boucle do-while est de pouvoir exécuter au moins une fois les instructions du corps avant de tester la condition d’arrêt. Cela peut éviter la duplication des instructions de modification des variables du test, qui est parfois nécessaire dans les boucles while.

Exemple 

Terminal.writeString("Entrez un entier (terminer le programme avec 0): ");
n = Terminal.readInt(); // Saisie d’initialisation pour n
total = 0;

while ( n != 0 )
{
  total = total + n;
  Terminal.writeString("Entrez un entier (terminer le programme avec 0): ");
  n = Terminal.readInt(); // Nouvelle saisie de n }
}

Exercice

La notion de bloc

  • Un bloc est une suite d'insctructions contenues entre deux accolades
  • La vie d'une variable (portée) est liée à un bloc
C'est le cas pour les conditions, les boucles, ou encore la méthode main

Exemple 

public static void main (String[] args) {
  /*debut bloc 1*/
  int a = 2;
  Terminal.writeStringln("valeur de a: "+ a );
  if (a == 0) {
    /* debut bloc 2 */
    int b = 3 + a;
    Terminal.writeStringln("valeur de b: "+ b );
    /* fin bloc 2*/
  } else {
    /* debut bloc 3*/
    int c = 3 + a;
    Terminal.writeStringln("valeur de c: "+ c);
    /* fin bloc 3*/
  }
  Terminal.writeStringln("valeur de a: "+ a );
  /* fin bloc 1*/
}
